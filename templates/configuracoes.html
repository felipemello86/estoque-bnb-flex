Endereço onde a Evolution API está rodando.
Formato: código do país + DDD + número (ex: 5511999999999)
Como configurar a Evolution API
  1. Instale a Evolution API no servidor: docker run -p 8080:8080 atendai/evolution-api
  2. Acesse o painel da Evolution API e crie uma instância (ex: bnbflex)
  3. Conecte o WhatsApp escaneando o QR Code gerado
  4. Copie a API Key e preencha os campos acima
  5. Clique em Testar Conexão para verificar se está funcionando
